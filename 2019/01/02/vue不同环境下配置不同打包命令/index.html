vue不同环境下配置不同打包命令

发表于 2019-01-02
本文分步骤给大家介绍vue给不同环境配置不同打包命令的方法，非常不错，具有一定的参考借鉴价值,需要的朋友可以参考下...

实际工作当中，我们再打包上生产和测试的时候，是有不同的apiurl接口地址的。这个时候我们当然可以在每次打包的时候手动更改apiurl,也不是很麻烦
不过，如果有更炫酷的操作，比如打包时指定不同的命令参数，就自动打包出不同环境的包，岂不是更好嘛。

话不多说，实战干活直接上。

1、安装cross-env

cross-env是个什么东西呢，自行百度吧（我也是百度了看了下，大概就是一个解决windows下不支持配置node进程process NODE_ENV的东西，有了它就可以跨平台设置及使用环境变量）

npm i cross-env -D
